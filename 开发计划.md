# Always Display - 开发计划文档

## 项目概述

| 项目 | 说明 |
|------|------|
| 应用名称 | Always Display |
| 目标设备 | vivo X200s |
| 开发语言 | Kotlin |
| UI框架 | Jetpack Compose |
| 最低API | Android 8.0 (API 26) |
| 配色风格 | 黑白温色调（OLED省电） |
| 核心特色 | 类原子岛平台音乐歌词+封面实时显示 |

## 一、功能需求

### 1.1 时钟模块

| 功能 | 说明 |
|------|------|
| 当前时间 | 大字体HH:MM:SS显示，每秒更新 |
| 日期显示 | YYYY年MM月DD日 + 星期X |
| 横屏适配 | 全屏大字体，居中显示 |
| 切换方式 | 左滑切换到倒计时，上滑切换到音乐 |

### 1.2 倒计时模块

| 功能 | 说明 |
|------|------|
| 时间设置 | 滚轮选择器：时(0-23) 分(0-59) 秒(0-59) |
| 快捷预设 | 1分钟 / 5分钟 / 10分钟 / 25分钟(番茄钟) / 自定义 |
| 最大时长 | 24小时 |
| 控制按钮 | 开始 / 暂停 / 重置 |
| 结束提醒 | 震动 + 嘀嘀嘀声音 |
| 显示方式 | 大字体倒计时 + 进度条(可拖动调整) |
| 切换方式 | 右滑切换到时钟，上滑切换到音乐 |

### 1.3 平台音乐监听模块（核心新增）

| 功能 | 说明 |
|------|------|
| 监听机制 | NotificationListenerService 监听系统媒体通知 |
| 数据来源 | 任意音乐App（网易云、QQ、Spotify、Apple Music等） |
| 封面获取 | MediaSession.Metadata 中的 METADATA_KEY_ART / ART_URI |
| 歌曲信息 | 歌名、歌手、专辑名、时长、播放进度 |
| 播放状态 | 播放/暂停/切歌 实时同步 |
| 歌词来源 | 在线歌词API（根据歌名+歌手匹配） |
| 歌词显示 | 原子岛紧凑模式 + 全屏歌词模式 |
| 封面显示 | 原子岛小封面 + 全屏大封面 |
| 播放控制 | 上一首/播放暂停/下一首（通过MediaController发送指令） |
| 进度显示 | 原子岛进度条 + 全屏进度条(可拖动) |
| 权限引导 | 首次使用引导用户开启通知监听权限 |

### 1.4 本地音乐模块（保留）

| 功能 | 说明 |
|------|------|
| 音乐来源 | 本地音乐文件 |
| 数据获取 | MediaStore API扫描 |
| 唱片动画 | 左侧半圆唱片，只显示一半 |
| 封面来源 | 音乐文件内嵌专辑封面 |
| 旋转效果 | 播放时顺时针旋转 |
| 切换动画 | 切换歌曲时有过渡动画 |
| 歌词来源 | 本地LRC文件 + 在线歌词API回退 |
| 歌词显示 | 右侧滚动显示，不显示时间戳 |
| 高亮效果 | 当前播放行高亮显示 |
| 播放控制 | 上一首 / 播放暂停 / 下一首 |
| 播放模式 | 单曲循环 / 列表循环 / 随机播放 |
| 音量控制 | 硬件音量键联动 |

### 1.5 设置模块

| 功能 | 说明 |

### 2.1 核心架构：平台音乐监听

```

┌─────────────────────────────────────────────────┐

│  │ onNotificationPosted()                    │   │

│  │  → 注册 Callback 监听变化                 │   │

┌─────────────────────────────────────────────────┐
│            MediaSessionBridge                    │
│  ┌──────────────┐  ┌────────────────────────┐   │
│  │ 封面Bitmap    │  │ 歌名/歌手/时长/进度    │   │
│  └──────────────┘  └────────────────────────┘   │
│  ┌──────────────────────────────────────────┐    │
│  │ 播放控制: play/pause/skipToNext/seekTo   │    │
│  └──────────────────────────────────────────┘    │
└────────────────────┬────────────────────────────┘

              ┌──────┴──────┐
              ▼             ▼
     ┌────────────┐  ┌────────────┐
     │ 在线歌词API │  │ MusicVM    │
     │ 歌名+歌手   │  │ 统一状态   │
     │ → LRC歌词   │  │ 管理       │
     └─────┬──────┘  └─────┬──────┘
           │               │
           └───────┬───────┘
                   ▼
          ┌────────────────┐
          │   UI 渲染       │
          │ 原子岛 / 全屏   │
          └────────────────┘
```

### 2.4 在线歌词API方案

| 方案 | 说明 | 优缺点 |
|------|------|--------|
| 网易云音乐API（非官方） | 通过歌名+歌手搜索歌词 | 歌词库大，但非官方接口可能不稳定 |
| QQ音乐API（非官方） | 通过歌名+歌手搜索歌词 | 覆盖广，同样非官方 |
| LRCLIB (lrclib.net) | 开源歌词数据库，提供免费API | 合规、稳定、有REST API，推荐首选 |
| 本地LRC回退 | 本地匹配同名.lrc文件 | 无网络时使用 |

**推荐方案**：LRCLIB 作为主歌词源 + 本地LRC回退

```
GET https://lrclib.net/api/search?q={songName}+{artistName}
返回 JSON 包含同步歌词 (syncedLyrics) 和纯文本歌词 (plainLyrics)
```

### 3.1 颜色定义

| 名称 | 色值 | 用途 |
|------|------|------|
| 背景色 | #FF000000 | 主背景（OLED纯黑，省电） |

| 原子岛背景 | #FF1C1C1E | 原子岛胶囊背景（微透明感） |

| 副文字色 | #FFB0B0B0 | 次要文字 |

| 暗灰色 | #FF666666 | 普通歌词 |

### 3.2 省电设计

- 使用OLED纯黑背景（#FF000000），黑色像素完全不发光
- 避免大面积白色，文字使用灰度区分层级
- 歌词只高亮当前行，其余行暗灰
- 原子岛模式下最小化UI面积，减少发光像素

## 六、省电优化策略

### 6.1 屏幕管理

| 场景 | 策略 |
|------|------|
| 屏幕常亮开启 | 使用FLAG_KEEP_SCREEN_ON |
| 屏幕常亮关闭 | 跟随系统息屏设置 |
| 亮度控制 | 设置中提供降低屏幕亮度选项 |

### 6.2 Compose渲染优化

| 技术 | 实现方式 |
|------|----------|
| 1Hz刷新 | LaunchedEffect + 对齐系统时钟的delay |
| 最小化重组 | derivedStateOf + 精确状态控制 |
| 静态UI冻结 | Modifier.drawWithCache |
| 原子岛模式 | 仅更新歌词行+进度条，其余UI不重组 |

### 6.3 动画优化

| 动画类型 | 策略 |
|----------|------|
| 唱片旋转 | 仅播放时旋转，暂停时保持当前角度 |
| 封面切换 | Animatable + snapTo，一次性过渡 |
| 歌词滚动 | 按需更新，debounce防抖 |
| 原子岛展开 | 简单尺寸动画，避免复杂过渡 |

### 6.4 网络优化（新增）

| 优化项 | 实现方式 |
|--------|----------|
| 歌词缓存 | Room数据库缓存已获取的歌词，避免重复请求 |
| 请求合并 | 同一歌曲不重复请求歌词API |
| 离线回退 | 优先本地LRC → 缓存 → 在线API |
| 请求超时 | 5秒超时，失败静默降级 |

### 6.5 预估耗电

| 场景 | 预估耗电 |
|------|----------|
| 纯时钟显示 | ~0.5%/小时 |
| 倒计时运行 | ~0.8%/小时 |

```kotlin
class MediaListenerService : NotificationListenerService() {

    private val bridge = MediaSessionBridge()

    override fun onListenerConnected() {
        super.onListenerConnected()
        // 从已有通知中恢复媒体会话
        activeNotifications?.forEach { sbn ->
            extractMediaSession(sbn)
        }
    }

    override fun onNotificationPosted(sbn: StatusBarNotification, rankingMap: RankingMap?) {
        extractMediaSession(sbn)
    }

    override fun onNotificationRemoved(sbn: StatusBarNotification, rankingMap: RankingMap?) {
        val packageName = sbn.packageName
        bridge.removeSession(packageName)
    }

    private fun extractMediaSession(sbn: StatusBarNotification) {
        val token = sbn.notification.extras.getParcelable(
            Notification.EXTRA_MEDIA_SESSION,
            MediaSession.Token::class.java
        ) ?: return

        bridge.connect(this, sbn.packageName, token)
    }
}
```

### 10.2 MediaSession桥接层（核心新增）

```kotlin
class MediaSessionBridge {

    private val controllers = mutableMapOf<String, MediaController>()
    private val _mediaState = MutableStateFlow<MediaInfo?>(null)
    val mediaState: StateFlow<MediaInfo?> = _mediaState.asStateFlow()

    data class MediaInfo(
        val packageName: String,
        val title: String,
        val artist: String,
        val album: String,
        val duration: Long,
        val position: Long,
        val isPlaying: Boolean,
        val coverBitmap: Bitmap?,
        val controller: MediaController
    )

    fun connect(context: Context, packageName: String, token: MediaSession.Token) {
        if (controllers.containsKey(packageName)) return

        val controller = MediaController(context, token)
        controller.registerCallback(object : MediaController.Callback() {
            override fun onMetadataChanged(metadata: MediaMetadata?) {
                updateState(packageName, controller)
            }

            override fun onPlaybackStateChanged(state: PlaybackState?) {
                updateState(packageName, controller)
            }
        })
        controllers[packageName] = controller
        updateState(packageName, controller)
    }

    fun removeSession(packageName: String) {
        controllers.remove(packageName)?.unregisterCallback(null)
        // 如果移除的是当前活跃会话，清空状态
        if (_mediaState.value?.packageName == packageName) {
            _mediaState.value = null
        }
    }

    private fun updateState(packageName: String, controller: MediaController) {
        val metadata = controller.metadata ?: return
        val state = controller.playbackState ?: return

        _mediaState.value = MediaInfo(
            packageName = packageName,
            title = metadata.getString(MediaMetadata.METADATA_KEY_TITLE) ?: "",
            artist = metadata.getString(MediaMetadata.METADATA_KEY_ARTIST) ?: "",
            album = metadata.getString(MediaMetadata.METADATA_KEY_ALBUM) ?: "",
            duration = metadata.getLong(MediaMetadata.METADATA_KEY_DURATION),
            position = state.position,
            isPlaying = state.state == PlaybackState.STATE_PLAYING,
            coverBitmap = metadata.getBitmap(MediaMetadata.METADATA_KEY_ART),
            controller = controller
        )
    }

    // 播放控制
    fun play() { _mediaState.value?.controller?.transportControls?.play() }
    fun pause() { _mediaState.value?.controller?.transportControls?.pause() }
    fun skipToNext() { _mediaState.value?.controller?.transportControls?.skipToNext() }
    fun skipToPrevious() { _mediaState.value?.controller?.transportControls?.skipToPrevious() }
    fun seekTo(pos: Long) { _mediaState.value?.controller?.transportControls?.seekTo(pos) }
}
```

```kotlin
class LyricRepository(
    private val apiService: LyricApiService,
    private val context: Context
) {
    private val cache = mutableMapOf<String, List<LyricLine>>()

    suspend fun getLyrics(title: String, artist: String): List<LyricLine> {
        val key = "${title}_${artist}"

        // 1. 内存缓存
        cache[key]?.let { return it }

        // 2. 本地LRC文件
        findLocalLrc(title, artist)?.let {
            val parsed = LyricParser.parse(it)
            cache[key] = parsed
            return parsed
        }

        // 3. 在线API
        try {
            val result = apiService.searchLyrics("$title $artist")
            if (result != null) {
                val parsed = LyricParser.parse(result.syncedLyrics ?: result.plainLyrics)
                cache[key] = parsed
                return parsed
            }
        } catch (e: Exception) {
            // 静默失败，返回空歌词
        }

        return emptyList()
    }

    private fun findLocalLrc(title: String, artist: String): InputStream? {
        // 尝试在音乐同目录下查找同名.lrc文件
        return null
    }
}
```

```kotlin
@Composable
fun VinylDisc(
    coverBitmap: Bitmap?,
    isPlaying: Boolean,
    modifier: Modifier = Modifier
) {
    var currentRotation by remember { mutableFloatStateOf(0f) }
    val rotationAnim = remember { Animatable(currentRotation) }

    // 播放时持续旋转，暂停时保持当前角度
    LaunchedEffect(isPlaying) {
        if (isPlaying) {
            rotationAnim.animateTo(
                targetValue = currentRotation + 360f,
                animationSpec = infiniteRepeatable(
                    animation = tween(durationMillis = 8000, easing = LinearEasing),
                    repeatMode = RepeatMode.Restart
                )
            ) {
                currentRotation = value
            }
        } else {
            rotationAnim.snapTo(currentRotation)
        }
    }

    Box(
        modifier = modifier
            .aspectRatio(1f)
            .clip(CircleShape)
            .graphicsLayer {
                rotationZ = rotationAnim.value
            }
    ) {
        // 封面图片
        coverBitmap?.let {
            Image(
                bitmap = it.asImageBitmap(),
                contentDescription = "专辑封面",
                modifier = Modifier.fillMaxSize()
            )
        }

        // 唱片纹路叠加
        Canvas(modifier = Modifier.fillMaxSize()) {
            val center = Offset(size.width / 2, size.height / 2)
            for (i in 1..8) {
                drawCircle(
                    color = Color.Black.copy(alpha = 0.15f),
                    radius = size.minDimension * i / 10f,
                    center = center,
                    style = Stroke(width = 1.dp.toPx())
                )
            }
            // 中心圆
            drawCircle(
                color = Color(0xFF333333),
                radius = size.minDimension * 0.08f,
                center = center
            )
        }
    }

    // 半圆裁切（在父容器中处理）
    Box(
        modifier = Modifier
            .fillMaxHeight()
            .width(50.dp)
            .align(Alignment.CenterEnd)
            .background(Color.Black)
    )
}
```

```kotlin
@Composable
fun LyricView(
    lyrics: List<LyricLine>,
    currentLineIndex: Int,
    modifier: Modifier = Modifier
) {
    val listState = rememberLazyListState()
    val density = LocalDensity.current
    val itemHeight = with(density) { 48.dp.toPx() }

    // 居中偏移：让当前行显示在列表中央
    val centerOffset = with(density) {
        -(LazyListState.layoutInfo.viewportSize.height / 2 - itemHeight / 2).toInt()
    }

    // 防抖：避免快速切歌时动画堆积
    LaunchedEffect(currentLineIndex) {
        if (currentLineIndex >= 0 && currentLineIndex < lyrics.size) {
            listState.animateScrollToItem(
                index = currentLineIndex,
                scrollOffset = centerOffset
            )
        }
    }

    LazyColumn(
        state = listState,
        modifier = modifier,
        horizontalAlignment = Alignment.CenterHorizontally,
        contentPadding = PaddingValues(
            vertical = with(density) {
                (listState.layoutInfo.viewportSize.height / 2).toDp()
            }
        )
    ) {
        itemsIndexed(lyrics) { index, line ->
            val isCurrent = index == currentLineIndex
            val animColor by animateColorAsState(
                targetValue = if (isCurrent) Color.White else Color(0xFF666666),
                animationSpec = tween(300)
            )
            val animSize by animateFloatAsState(
                targetValue = if (isCurrent) 20f else 16f,
                animationSpec = tween(300)
            )

            Text(
                text = line.text,
                color = animColor,
                fontWeight = if (isCurrent) FontWeight.Bold else FontWeight.Normal,
                fontSize = animSize.sp,
                textAlign = TextAlign.Center,
                modifier = Modifier.padding(vertical = 8.dp)
            )
        }
    }
}
```

```kotlin
class SoundPlayer(private val context: Context) {
    private val soundPool = SoundPool.Builder()
        .setMaxStreams(1)
        .setAudioAttributes(
            AudioAttributes.Builder()
                .setUsage(AudioAttributes.USAGE_ALARM)
                .setContentType(AudioAttributes.CONTENT_TYPE_SONIFICATION)
                .build()
        )
        .build()

    private var beepSoundId: Int = 0
    private var isLoaded = false

    init {
        soundPool.setOnLoadCompleteListener { _, _, status ->
            isLoaded = status == 0
        }
        beepSoundId = soundPool.load(context, R.raw.beep, 1)
    }

    fun playBeep() {
        if (isLoaded) {
            soundPool.play(beepSoundId, 1f, 1f, 1, 0, 1f)
        }
    }

    // 连续嘀嘀嘀
    fun playBeepRepeat(times: Int = 3, intervalMs: Long = 500) {
        if (!isLoaded) return
        CoroutineScope(Dispatchers.Main).launch {
            repeat(times) {
                soundPool.play(beepSoundId, 1f, 1f, 1, 0, 1f)
                delay(intervalMs)
            }
        }
    }

    fun release() {
        soundPool.release()
    }
}
```

```kotlin
                REQUEST_AUDIO

        }
```

## 十二、注意事项

1. **通知监听权限**：需引导用户手动开启，无法动态申请
2. **MediaSession兼容性**：不同音乐App暴露的Metadata字段可能不同，需做容错
3. **封面获取**：部分App通过URI而非Bitmap提供封面，需Coil异步加载
4. **歌词匹配**：在线歌词可能匹配不准，需支持手动搜索
5. **权限处理**：需要运行时请求READ_MEDIA_AUDIO权限
6. **兼容性**：适配Android 8.0-15
7. **横竖屏**：支持屏幕旋转
8. **后台播放**：本地音乐使用MediaSessionService
9. **资源释放**：页面退出时释放ExoPlayer
10. **错误处理**：无音乐时显示提示，网络失败时静默降级
11. **空状态**：无歌词时显示"暂无歌词"
12. **隐私合规**：通知监听属敏感权限，需在隐私政策中说明用途
13. **Android 14+**：前台服务需声明 foregroundServiceType
